# Playwright Hub

App tổng hợp kiến thức Playwright cho người mới bắt đầu, có search lệnh nhanh.

## Tính năng

- 12 chương từ cơ bản đến CI/CD
- Search lệnh thông minh: gõ tên lệnh → nhảy đến đúng mục giải thích
- Phân loại theo category: locator / action / assertion / network...
- Code TypeScript có syntax highlight, copy bằng 1 click
- Phím tắt `Cmd+K` / `Ctrl+K` để mở search

## Chạy local

Mở `index.html` bằng trình duyệt là xong, không cần build.

## Deploy

Đã deploy trên Vercel: tự động cập nhật khi push code mới lên branch `main`.
